# Events Library Templates

This repository provides boilerplate templates for integrating the **[Events](https://github.com/a8kj7sea/Events)** library into your Spigot plugins using either **Maven** or **Gradle (Kotlin DSL)**.


## Project Structure

* `/maven`: Template project using standard `pom.xml`.
* `/gradle`: Template project using `build.gradle.kts` and Shadow plugin.
